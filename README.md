# A [BrainFuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter written in C.

## Example

##### Source File

```brainfuck
++++++++++[>+++++++>++++++++++>+++++++++++>++++>+++>++++++++++++>++++++++++>+++++++++++<<<<<<<<-]>++.>+.>--..+++.>++++.>++.>----.>++++.<<<<<.>>>>>>++++.<<<<<<.>>>+.
```

##### Terminal

```
./brainfuck helloThere.bf
> Hello, there!
```
